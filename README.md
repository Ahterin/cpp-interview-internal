# C++ Interview Internal Pack

This repository is for interviewers only.

## Key files

- `PREPARE_INTERVIEW.md`: pre-interview checklist and candidate handoff.
- `INTERVIEWER_GUIDE.md`: exercise intent and expected findings.
- `FOLLOWUP_QUESTIONS.md`: prompts for discussion during interview.
- `SCORING_TEMPLATE.md`: template for candidate scoring and notes.
- `REFERENCE_SOLUTION.md`: reference change set and verification command.

## Scripts

- `scripts/prepare_candidate.sh <firstname-lastname>`: creates a fresh candidate branch from `candidate/task-price-fix`.
- `scripts/verify_reference_solution.sh`: validates that the reference changes still build and pass tests.
