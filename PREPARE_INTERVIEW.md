# Prepare Interview

## Before the interview
1. Make sure your local repo is clean.
2. Make sure the candidate base branch exists.
3. Create a candidate branch with `./scripts/prepare_candidate.sh <firstname-lastname>`.
4. Push it if needed.

## What to give the candidate
- repository URL
- branch name
- authentication method
- instruction to start from README
- note that AI/search use is allowed, but they must understand and explain their changes
